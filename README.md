Code Villain – HTML Poster Project

 About the Project

This is a small HTML project that displays a “Wanted” poster for a fictional coding villain called The Bug Master.
The poster shows the villain’s crimes, danger level, and reward. Clicking the image reveals a secret information page about the villain.

Purpose

This project was created to practice basic HTML and CSS skills,

Features

-  Clickable image leading to a hidden page
-  List of the villain's coding crimes
-  Reward section and contact information
-  Multiple connected HTML pages
